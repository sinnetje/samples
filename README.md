# Code samples
This repository contains code samples from various projects I've worked on, and is for demo purposes only.

[ Afsprakenkalender ]
---------------------
* Sample of a MeteorJS project.
* Used for displaying a whole week in a calendar (Mon-Sun, with hours on the y-axis).
* Including the display of appointments and the option to accept/decline/edit/delete them.

- week.html:             The html templates for the calendar week overview (Mon-Sun, per hour).
- week.js:               The data and events used in the templates of week.html.
- appointmentsSchema.js: The appointment data model.



[ Planningssysteem ]
----------------------------
* Sample of a PHP project.
* Used for displaying an input form for adding holidays to the system.
* Custom holidays (i.e. depending on religion) or standard holidays (e.g. Christmas) can be added.
* Written in a custom framework (hence the .egg file).

- HolidaysForm.php:     Input form for adding holidays to the system.
- PopupFormView.php:    The template class extended by HolidaysForm.php.
- holidays.egg:         External file from which the standard holiday dates are loaded.
- SaveHoliday.php:      The action script for saving holiday data.
- mixins.scss:          Example of mixins written in .scss.
